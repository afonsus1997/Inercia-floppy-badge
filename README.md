## Specifications and Features

- **Processor:** RP2040
- **Memory:** 264kB on-chip SRAM, 32Mb external flash
- **Power Supply:** USB or 3xAAA
- **Connectivity:** USB, external GPIO, SAO
- **Dimensions:** Very close to a real floppy!
